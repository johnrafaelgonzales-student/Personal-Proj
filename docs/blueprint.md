# **App Name**: LibFlow

## Core Features:

- Manual Visitor Entry Logging: A user-friendly form for librarians to quickly log new visitor entries, capturing date, time, and purpose of visit.
- RF ID Self-Service Entry: Allows authorized users (students, faculty, employees) to log their entry by tapping their NEU School ID.
- Email Self-Service Entry: Provides an alternative self-service entry method for users by allowing them to log in using their institutional email (Google-based).
- Entry Point Information Display: A dedicated terminal interface that displays the entrant's name, college/office upon successful entry, prompts for their specific purpose of visit, and shows a welcome message: "Welcome to NEU Library!".
- Automatic Timestamping: Automatically records the current date and time upon any visitor entry submission (manual, RF ID, or email).
- Visitor Log Display: View a paginated list of all visitor entries, with options to sort and filter by date.
- Admin Dashboard Overview: A centralized dashboard for administrators to view key visitor statistics and trends, monitoring the number of visitors per day, per week, or a duration period set.
- Admin Entry Management: Enables administrators to view, manage, and block specific visitor entries through the application interface.
- Period-based Statistics: Generate reports showing visitor counts and statistics aggregated by day, week, or month.
- Custom Duration Reporting: Allow administrators to define a custom date range to analyze visitor statistics within that period.
- Visitor Trend Analysis Tool: An AI-powered tool that provides insights into visitor patterns, suggesting peak hours or days based on historical data.
- PDF Report Generation: Generate and download reports in PDF format for selected statistics.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#1E57AA), reflecting the institution's professionalism and knowledge.
- Background color: A very light, subtle blue-gray (#F0F4F7), creating a calm and clean digital environment.
- Accent color: A vibrant aqua-green (#4DBBAE), providing a refreshing contrast and highlighting interactive elements.
- Body and headline font: 'Inter' (sans-serif), chosen for its excellent readability, modern aesthetic, and neutrality, suitable for data-heavy applications and clean interfaces.
- Use clear and minimalistic line icons for navigation and actions, maintaining a professional and intuitive user experience. Examples include 'logbook' for entries, 'chart' for dashboard, 'calendar' for date selection.
- Adopt a clean, card-based layout for information display, providing distinct visual separation for different data points and sections. Prioritize logical flow for form inputs.
- Subtle and quick animations for form submissions and report generation to provide immediate feedback to the user without distracting from the core function.
